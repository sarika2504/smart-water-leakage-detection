💧 Smart Water Leakage Detection with Automated Alerts and Analytics

 📖 Overview
This project is an Arduino-based smart water leakage detection system that detects water leakage using dual flow sensors and automatically controls a water pump through a relay. The system displays real-time flow information on an LCD, helping reduce water wastage.

## ✨ Features
- Real-time water flow monitoring
- Water leakage detection
- Automatic pump control
- LCD status display
- Relay-based automation
- Low-cost embedded solution

## 🛠 Components Used
- Arduino UNO
- Water Flow Sensors (2)
- 16×2 LCD Display
- Relay Module
- Water Pump
- Breadboard
- Jumper Wires

## 💻 Software
- Arduino IDE
- Embedded C
